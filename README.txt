HOW TO USE:
1. Make sure yt-dlp is installed and in your system PATH.
   https://github.com/yt-dlp/yt-dlp#installation

2. Run the yt_music_lite.exe executable.

3. Paste a YouTube link or type a search query and press ENTER.

4. It will fetch the best audio stream and start playing.

SYSTEM REQUIREMENTS:
- Windows 10 or higher
- yt-dlp (installed)
- Network access to YouTube

NOTE: This app streams directly and does not download files.

CREDITS:
- Built in Rust using crossterm, rodio, and ureq crates.
